# 🚀 Quick Start

Instala dependencias:

```bash
pnpm install
```

Ejecuta el asistente de configuración:

```bash
pnpm run init-project
```

Inicia el servidor de desarrollo:

```bash
pnpm dev
```

---

# ⚙️ Personalización del Template

El comando:

```bash
pnpm run init-project
```

configura automáticamente:

* package.json
* nuxt.config.ts
* app/app.config.ts
* README.md
* .env
* .env.example

Durante el asistente se solicitará:

* Nombre del proyecto
* Descripción
* Título de la aplicación
* Color principal
* Color neutral
* Autor
* Inicialización de Git
